## 修复 blogData is not defined 错误

- **问题现象**：网站首页显示"首页加载失败：blogData is not defined"
- **根本原因**：`index.html` 中脚本加载顺序错误，`data.js` 在 CDN 脚本（marked、hljs）之后加载；如果 CDN 加载阻塞或失败，会影响后续脚本执行
- **修复内容**：
  1. 将 `js/data.js` 和 `js/router.js` 移至所有 CDN 脚本之前加载
  2. 固定 marked 版本为 `v4.3.0`（稳定 API，避免 v5 破坏性变更）
  3. 加入 CDN 失败降级处理：若 marked 未加载，提供简单 HTML 转换；若 hljs 未加载，提供空实现
  4. `app.js` 中对 `marked.parse()` 返回值做 Promise 判断兼容

- **推送状态**：网络连接 GitHub 失败，待网络恢复后执行 `git push origin main`
