## 📝 如何添加新文章

打开 `js/data.js` 文件，在 `articles` 数组中添加新文章对象：

```javascript
{
    id: 7,  // 确保ID唯一，递增
    title: "文章标题",
    excerpt: "文章摘要（显示在列表页）",
    content: `# 文章标题

## 小标题

正文内容支持 **Markdown** 格式：

- 列表项
- 列表项

> 引用块

\`\`\`javascript
// 代码块
console.log("Hello");
\`\`\`
`,
    date: "2026-04-03",  // 发布日期
    readTime: "8分钟",    // 预计阅读时间
    tags: ["标签1", "标签2", "标签3"],  // 文章标签
    image: "default"      // 图片标识（可选）
}
```

## 🏷️ 管理标签

- 标签在添加文章时直接写入 `tags` 数组
- 网站会自动统计所有标签并生成标签云
- 建议保持标签一致性，例如统一使用"心理健康"而非混用"心理"和"心理健康"


## 🔍 常见问题

### 1. 文章显示不出来？
- 检查 `id` 是否唯一
- 检查 JSON 语法（逗号、引号是否完整）
- 使用浏览器开发者工具查看控制台错误

### 2. Markdown 格式不正确？
- 确保使用模板字符串（反引号 `）包裹内容
- 代码块需要转义反引号：\`\`\`

### 3. 样式没有生效？
- 清除浏览器缓存（Ctrl+F5）
- 检查 CSS 文件路径是否正确
